## 🗺️ ¿Qué vamos a construir en estas 10 sesiones?

**Oriente Stereo** es una emisora comunitaria del Oriente de Cali. Durante años han grabado entrevistas, crónicas y debates del territorio. Ese archivo sonoro es memoria colectiva — no tiene precio.

**El problema real:** parte de ese archivo ya se perdió por daño físico de equipos. El resto sigue en riesgo. Los audios están repartidos en tres lugares distintos (computador, discos externos y Google Drive), sin orden, sin nombres uniformes, sin respaldo sistemático.

**Nuestra misión en 10 sesiones:** construir un sistema en Python que:

| Sesión | ¿Qué construimos? | ¿Qué aprendemos? |
|---|---|---|
| **S1** | Entorno + primeras líneas de Python | `print()`, variables, tipos de datos (`str`, `int`, `float`, `bool`), f-strings, operaciones básicas de strings |
| **S2** | Función para estandarizar nombres de archivos | Funciones (`def`, parámetros, `return`), condicionales (`if`/`elif`/`else`), operador `in`, `.replace()`, `.strip()` |
| **S3** | Función para detectar archivos de audio | Listas, bucles `for`, acumuladores, filtrado por condición |
| **S4** | Script para explorar carpetas en múltiples rutas | `pathlib`, `Path`, `.rglob()`, `.stat()`, `try`/`except`, recorrido recursivo de carpetas |
| **S5** | Catálogo CSV con todos los programas | Módulo `csv`, `DictWriter`, `DictReader`, campos opcionales, encoding `utf-8-sig`, Pandas |
| **S6** | Módulo con manejo de errores | Estructura de módulos Python, `import`, `except` con tipos específicos, `docstrings` |
| **S7** | Lógica de priorización del backup | `sorted()` con `key=`, acumuladores con condición, operaciones con diccionarios |
| **S8** | Backup a Google Drive | Variables de entorno (`os.environ`), autenticación con APIs, manejo de errores de red |
| **S9** | Backup a AWS S3 (segunda capa) | Librería `boto3`, cliente S3, manejo de errores de conexión y región |
| **S10** | Sistema completo + entrega a Oriente Stereo | Integración de módulos, `sys.path`, CLI, README técnico, credenciales en variables de entorno |
